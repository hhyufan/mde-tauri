# 🎉 MDE v0.10 发布 - 多模式 Markdown 工作台与云同步

[![release](https://img.shields.io/badge/release-v0.10-green)](https://github.com/hhyufan/mde-tauri) [![date](https://img.shields.io/badge/date-2026--06--09-blue)](https://github.com/hhyufan/mde-tauri)

## 🚀 版本亮点

### 📝 多模式 Markdown 编辑工作台正式成型

这是 MDE 的一个阶段性版本。`v0.10` 让项目从“基础 Markdown 编辑器”进一步演进为一套完整的 Markdown 工作台，围绕 **本地文件管理、源码编辑、所见即所得编辑、实时预览、搜索导航、云同步与设置中心** 做了整体串联。

本版本的核心目标是把“写、看、找、存、同步”这些高频动作放到一套统一界面里完成，让 Markdown 编辑不再只是单纯输入文本，而是升级为一套更完整的知识整理与文档创作体验。

### ✨ 核心新特性

- **🧭 资源管理器工作流**：支持打开目录、文件树浏览、前进/后退、上一级、刷新、排序、面包屑导航、在系统资源管理器中定位
- **📝 多模式编辑**：支持 Monaco 源码编辑、Milkdown 所见即所得编辑、Markdown 实时预览、分屏编辑/预览
- **🔍 全局搜索与导航**：支持文件名搜索、Markdown 内容搜索、命中跳转、大纲联动跳转
- **🔗 智能 Markdown 链接**：支持相对路径链接在应用内打开，并支持 `#Lxx` / `Lxx~xx` 行号跳转提示
- **📖 增强 Markdown 渲染**：支持 GFM、表格、任务列表、脚注、KaTeX 数学公式、Mermaid 图表、代码高亮、部分 HTML 渲染
- **☁️ 云同步能力**：支持账号登录、文档同步、云端仅文档、冲突对比解决、设置同步、配置导入导出
- **🎨 界面与交互升级**：支持亮暗主题切换、View Transition 动画、浮动工具栏、状态栏信息、桌面/移动端响应式适配
- **📂 拖拽与多标签工作流**：支持拖拽打开文件、拖拽移动到资源管理器目录、多标签页切换与未保存状态保护

## 📝 功能详解

### 🧭 资源管理器与文件管理

- **目录级工作流**：现在可以直接打开文件夹，在侧边栏中浏览整个工作目录
- **导航能力补全**：支持前进、后退、上一级、刷新、关闭目录、长路径面包屑导航
- **排序能力**：支持按名称、时间、大小排序，并支持升序 / 降序切换
- **文件操作完善**：支持新建文件、打开文件、保存、另存为、重命名、删除
- **系统联动**：桌面端支持在系统资源管理器中直接定位当前目录或文件

### ✍️ 多模式编辑体验

- **Monaco 源码模式**：适合偏代码风格的 Markdown 编写，具备熟悉的源码编辑体验
- **Milkdown 所见即所得**：支持更直观的可视化编辑，对任务列表、表格、图片、公式、代码块等内容提供更友好的交互
- **实时预览模式**：单独预览视图可直接查看 Markdown 最终渲染效果
- **分屏模式**：编辑与预览可同时展示，方便边写边看
- **浮动工具栏**：常见 Markdown 语法可通过工具栏快速插入，降低记忆负担

### 🔍 搜索与结构化导航

- **全局搜索**：支持按文件名搜索，也支持 Markdown 内容搜索
- **命中跳转**：内容搜索结果可直接打开文件并跳到对应位置
- **大纲视图**：自动提取标题与结构信息，便于快速定位长文档内容
- **最近文件 / 书签 / 云文档聚合**：一个入口同时覆盖本地与云端工作流

### 🔗 Markdown 链接与跳转体验

- **应用内打开相对路径**：Markdown 文档中的相对路径链接可直接在 MDE 内部新标签页打开
- **本地与外部链接分流**：本地路径优先进入编辑器，外部链接交给系统浏览器
- **代码行号提示跳转**：支持通过链接 hash 或文本提示识别 `#Lxx`、`Lxx~xx` 这样的定位信息
- **预览区与所见即所得统一行为**：Markdown 预览与 Milkdown 中的链接跳转体验保持一致

### 📖 Markdown 渲染引擎增强

- **GFM 兼容**：支持表格、任务列表、删除线、脚注等常见扩展语法
- **数学公式支持**：集成 KaTeX，适合技术文档和学习笔记场景
- **Mermaid 图表渲染**：支持流程图、时序图等图表内容
- **代码高亮与复制**：预览区代码块支持语言标识、高亮与一键复制
- **图片与部分 HTML 支持**：增强 Markdown 内容承载能力，适合更复杂的文档表现

### ☁️ 云同步与账号系统

- **账号登录 / 注册**：支持 JWT 鉴权的云端账户体系
- **文档同步**：本地文档可与云端建立绑定并参与同步
- **云端仅文档**：支持仅存在于云端、尚未落地到本地路径的文档打开方式
- **冲突解决**：发生冲突时提供双栏比对，可明确选择保留本地或采用远端
- **设置同步**：编辑器设置、主题、布局等可同步到云端
- **配置导入导出**：支持设置 JSON 导入 / 导出，方便迁移环境

### 🎨 界面与交互优化

- **亮暗主题切换**：支持主题持久化，并通过 View Transition 增强切换体验
- **桌面式布局**：标题栏、标签栏、侧边栏、状态栏分工更明确
- **移动端适配**：在小屏设备上自动切换为更紧凑的布局
- **拖拽交互**：文件可直接拖入编辑区打开，也可拖入资源管理器完成移动与导入
- **未保存保护**：关闭窗口前可勾选需要保存的标签，降低误操作风险

## 🔧 本版本适合谁

- **Markdown 重度写作者**：需要更稳定、更完整的本地 Markdown 工作台
- **技术文档用户**：需要代码高亮、Mermaid、数学公式、内部链接跳转
- **知识整理用户**：需要目录管理、大纲导航、最近文件、书签与搜索能力
- **多设备协作者**：需要登录、同步、冲突解决与设置迁移能力

## 📌 版本总结

`v0.10` 的重点不是单一功能点，而是把多个核心模块真正连成一套统一体验：

- 文件管理不再只是“打开单个文件”，而是完整工作目录管理
- Markdown 不再只有源码输入，而是源码 / WYSIWYG / 预览三种模式协同
- 搜索、跳转、大纲、最近文件与云同步形成闭环
- 桌面端与移动端开始具备更清晰的适配策略

这意味着 MDE 已经从“可用的编辑器原型”逐步进入“可持续迭代的完整 Markdown 应用”阶段。
