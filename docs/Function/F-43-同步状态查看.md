# 同步状态查看

## 功能是什么

展示当前同步状态与结果。

## 对应代码在哪

- 代码位置：[syncEngine.js](../../../src/services/syncEngine.js)
  符号：`onStatusChange / setStatus`
  作用：同步引擎的状态源头。
- 代码位置：[useSyncStore.js](../../../src/store/useSyncStore.js)
  符号：`setLastSyncError / docs state`
  作用：保存文档级同步状态和最近错误。
- 代码位置：[SyncStatusIndicator.jsx](../../../src/components/ui/SyncStatusIndicator.jsx)
  符号：`SyncStatusIndicator`
  作用：把状态映射为 UI 图标和提示文案。

## 关键代码怎么读

- 第一步看 [syncEngine.js](../../../src/services/syncEngine.js) 中的 `onStatusChange / setStatus`，其作用是：同步引擎的状态源头。
- 第二步看 [useSyncStore.js](../../../src/store/useSyncStore.js) 中的 `setLastSyncError / docs state`，其作用是：保存文档级同步状态和最近错误。
- 第三步看 [SyncStatusIndicator.jsx](../../../src/components/ui/SyncStatusIndicator.jsx) 中的 `SyncStatusIndicator`，其作用是：把状态映射为 UI 图标和提示文案。
