# 大纲视图

## 功能是什么

自动识别标题层级并支持点击跳转，帮助长文档结构化导航。

## 对应代码在哪

- 代码位置：[OutlineView.jsx](../../../src/components/layout/sidebar/outline/OutlineView.jsx)
  符号：`extractItems / OutlineView`
  作用：从 Markdown 文本提取标题和列表，构造大纲侧栏。
- 代码位置：[MonacoEditor.jsx](../../../src/components/editor/MonacoEditor.jsx)
  符号：`handleOutlineJump`
  作用：收到大纲跳转事件后按行定位到源码。
- 代码位置：[MarkdownPreview.jsx](../../../src/components/editor/MarkdownPreview.jsx)
  符号：`outline:jump listener`
  作用：预览模式下按标题 slug 或列表文本滚动定位。
- 代码位置：[MilkdownMarkdownEditor.jsx](../../../src/components/editor/MilkdownMarkdownEditor.jsx)
  符号：`outline:jump listener`
  作用：WYSIWYG 模式下通过 DOM 或 ProseMirror 节点定位。

## 关键代码怎么读

- 第一步看 [OutlineView.jsx](../../../src/components/layout/sidebar/outline/OutlineView.jsx) 中的 `extractItems / OutlineView`，其作用是：从 Markdown 文本提取标题和列表，构造大纲侧栏。
- 第二步看 [MonacoEditor.jsx](../../../src/components/editor/MonacoEditor.jsx) 中的 `handleOutlineJump`，其作用是：收到大纲跳转事件后按行定位到源码。
- 第三步看 [MarkdownPreview.jsx](../../../src/components/editor/MarkdownPreview.jsx) 中的 `outline:jump listener`，其作用是：预览模式下按标题 slug 或列表文本滚动定位。
- 第四步看 [MilkdownMarkdownEditor.jsx](../../../src/components/editor/MilkdownMarkdownEditor.jsx) 中的 `outline:jump listener`，其作用是：WYSIWYG 模式下通过 DOM 或 ProseMirror 节点定位。
