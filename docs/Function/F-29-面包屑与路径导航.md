# 面包屑与路径导航

## 功能是什么

提供文件路径定位和快速层级切换能力。

## 对应代码在哪

- 代码位置：[FileTree.jsx](../../../src/components/layout/sidebar/explorer/FileTree.jsx)
  符号：`buildBreadcrumbPath / getParentDir`
  作用：根据当前目录生成面包屑路径。
- 代码位置：[FileTree.jsx](../../../src/components/layout/sidebar/explorer/FileTree.jsx)
  符号：`breadcrumbParts`
  作用：处理折叠、滚动和点击跳转逻辑。

## 关键代码怎么读

- 第一步看 [FileTree.jsx](../../../src/components/layout/sidebar/explorer/FileTree.jsx) 中的 `buildBreadcrumbPath / getParentDir`，其作用是：根据当前目录生成面包屑路径。
- 第二步看 [FileTree.jsx](../../../src/components/layout/sidebar/explorer/FileTree.jsx) 中的 `breadcrumbParts`，其作用是：处理折叠、滚动和点击跳转逻辑。
