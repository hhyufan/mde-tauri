# 最近文件

## 功能是什么

维护最近打开的文件列表，便于快速恢复工作状态。

## 对应代码在哪

- 代码位置：[useFileStore.js](../../../src/store/useFileStore.js)
  符号：`addRecentFile / clearRecentFiles / removeRecentFile`
  作用：维护最近文件列表。
- 代码位置：[Sidebar.jsx](../../../src/components/layout/sidebar/Sidebar.jsx)
  符号：`RecentList`
  作用：展示 recent，并允许直接打开或移除。
- 代码位置：[useFileManager.js](../../../src/hooks/useFileManager.js)
  符号：`openFileFromPath / finalizeSavedTab`
  作用：打开文件和保存文件都会写 recent。

## 关键代码怎么读

- 第一步看 [useFileStore.js](../../../src/store/useFileStore.js) 中的 `addRecentFile / clearRecentFiles / removeRecentFile`，其作用是：维护最近文件列表。
- 第二步看 [Sidebar.jsx](../../../src/components/layout/sidebar/Sidebar.jsx) 中的 `RecentList`，其作用是：展示 recent，并允许直接打开或移除。
- 第三步看 [useFileManager.js](../../../src/hooks/useFileManager.js) 中的 `openFileFromPath / finalizeSavedTab`，其作用是：打开文件和保存文件都会写 recent。
