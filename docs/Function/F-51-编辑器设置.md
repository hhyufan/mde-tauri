# 编辑器设置

## 功能是什么

支持 Tab 大小、编辑器行为、自动保存等参数。

## 对应代码在哪

- 代码位置：[SettingsModal.jsx](../../../src/components/overlays/SettingsModal.jsx)
  符号：`editor section`
  作用：负责字体、字号、Tab、大纲、自动换行、小地图等编辑器配置。
- 代码位置：[useConfigStore.js](../../../src/store/useConfigStore.js)
  符号：`editor config fields`
  作用：负责这些编辑器参数的本地保存。

## 关键代码怎么读

- 第一步看 [SettingsModal.jsx](../../../src/components/overlays/SettingsModal.jsx) 中的 `editor section`，其作用是：负责字体、字号、Tab、大纲、自动换行、小地图等编辑器配置。
- 第二步看 [useConfigStore.js](../../../src/store/useConfigStore.js) 中的 `editor config fields`，其作用是：负责这些编辑器参数的本地保存。
