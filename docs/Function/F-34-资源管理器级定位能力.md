# 资源管理器级定位能力

## 功能是什么

文件树、搜索、最近文件、书签与路径导航共同组成完整的文档定位体系。

## 对应代码在哪

- 代码位置：[FileTree.jsx](../../../src/components/layout/sidebar/explorer/FileTree.jsx)
  符号：`FileTree`
  作用：资源管理器级目录导航。
- 代码位置：[SearchModal.jsx](../../../src/components/overlays/SearchModal.jsx)
  符号：`SearchModal`
  作用：提供按文件名和内容定位的统一入口。
- 代码位置：[useFileStore.js](../../../src/store/useFileStore.js)
  符号：`recent / bookmarks / dir history`
  作用：为快速定位提供状态支撑。
- 代码位置：[Sidebar.jsx](../../../src/components/layout/sidebar/Sidebar.jsx)
  符号：`RecentList`
  作用：把 recent、bookmark、external docs 汇总展示。
