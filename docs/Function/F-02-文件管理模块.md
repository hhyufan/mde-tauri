# 文件管理模块

## 功能是什么

负责目录浏览、最近文件、书签、大纲和面包屑导航等文件组织能力。

## 对应代码在哪

- 代码位置：[useFileManager.js](../../../src/hooks/useFileManager.js)
  符号：`useFileManager`
  作用：负责文件系统相关操作的总编排。
- 代码位置：[useFileStore.js](../../../src/store/useFileStore.js)
  符号：`useFileStore`
  作用：负责当前目录、文件列表、最近文件、书签和排序状态。
- 代码位置：[FileTree.jsx](../../../src/components/layout/sidebar/explorer/FileTree.jsx)
  符号：`FileTree`
  作用：负责资源管理器式文件树和路径导航。
- 代码位置：[tauriApi.js](../../../src/utils/tauriApi.js)
  符号：`tauriApi`
  作用：负责把前端文件操作分发到桌面 Rust 层或 Android SAF。
- 代码位置：[lib.rs](../../../src-tauri/src/lib.rs)
  符号：`Tauri commands`
  作用：负责真实文件系统读写、重命名、删除、搜索和监听。

## 关键代码怎么读

- 第一步看 [useFileManager.js](../../../src/hooks/useFileManager.js) 中的 `useFileManager`，其作用是：负责文件系统相关操作的总编排。
- 第二步看 [useFileStore.js](../../../src/store/useFileStore.js) 中的 `useFileStore`，其作用是：负责当前目录、文件列表、最近文件、书签和排序状态。
- 第三步看 [FileTree.jsx](../../../src/components/layout/sidebar/explorer/FileTree.jsx) 中的 `FileTree`，其作用是：负责资源管理器式文件树和路径导航。
- 第四步看 [tauriApi.js](../../../src/utils/tauriApi.js) 中的 `tauriApi`，其作用是：负责把前端文件操作分发到桌面 Rust 层或 Android SAF。
- 最后回看其余支撑文件，补齐该功能的边界条件、状态变化和平台差异。
