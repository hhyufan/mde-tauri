# 设置与个性化模块

## 功能是什么

负责主题、字体、编辑器参数和同步配置管理。

## 对应代码在哪

- 代码位置：[SettingsModal.jsx](../../../src/components/overlays/SettingsModal.jsx)
  符号：`SettingsModal`
  作用：负责设置界面的分类展示和用户操作入口。
- 代码位置：[useConfigStore.js](../../../src/store/useConfigStore.js)
  符号：`useConfigStore`
  作用：负责通用、外观、编辑器和云同步设置的本地状态。
- 代码位置：[settingsSync.js](../../../src/utils/settingsSync.js)
  符号：`settingsSync`
  作用：负责生成和回放设置快照。
- 代码位置：[syncEngine.js](../../../src/services/syncEngine.js)
  符号：`setupConfigSubscriptions / syncConfig`
  作用：负责把本地设置变化同步到云端，或把远端设置拉回本地。
