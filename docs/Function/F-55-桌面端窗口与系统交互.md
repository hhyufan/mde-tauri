# 桌面端窗口与系统交互

## 功能是什么

桌面端支持窗口管理、拖拽打开、文件关联、快捷键操作和完整文件系统访问。

## 对应代码在哪

- 代码位置：[TitleBar.jsx](../../../src/components/layout/title-bar/TitleBar.jsx)
  符号：`window controls`
  作用：桌面端标题栏的最小化、最大化、关闭入口。
- 代码位置：[tauriApi.js](../../../src/utils/tauriApi.js)
  符号：`showInExplorer / showMainWindow / getCliArgs`
  作用：统一封装桌面系统交互。
- 代码位置：[lib.rs](../../../src-tauri/src/lib.rs)
  符号：`show_in_explorer / show_main_window / get_cli_args`
  作用：Rust 侧真正执行桌面系统调用。
- 代码位置：[App.jsx](../../../src/App.jsx)
  符号：`associated file open`
  作用：通过命令行参数支持双击 Markdown 文件直接打开。

## 关键代码怎么读

- 第一步看 [TitleBar.jsx](../../../src/components/layout/title-bar/TitleBar.jsx) 中的 `window controls`，其作用是：桌面端标题栏的最小化、最大化、关闭入口。
- 第二步看 [tauriApi.js](../../../src/utils/tauriApi.js) 中的 `showInExplorer / showMainWindow / getCliArgs`，其作用是：统一封装桌面系统交互。
- 第三步看 [lib.rs](../../../src-tauri/src/lib.rs) 中的 `show_in_explorer / show_main_window / get_cli_args`，其作用是：Rust 侧真正执行桌面系统调用。
- 第四步看 [App.jsx](../../../src/App.jsx) 中的 `associated file open`，其作用是：通过命令行参数支持双击 Markdown 文件直接打开。
