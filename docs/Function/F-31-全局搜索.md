# 全局搜索

## 功能是什么

提供统一搜索入口，支持关键词检索。

## 对应代码在哪

- 代码位置：[App.jsx](../../../src/App.jsx)
  符号：`Ctrl/Cmd+P handler`
  作用：提供全局搜索弹窗的快捷键入口。
- 代码位置：[SearchModal.jsx](../../../src/components/overlays/SearchModal.jsx)
  符号：`SearchModal`
  作用：负责搜索 UI、模式切换、结果选择和跳转。
- 代码位置：[tauriApi.js](../../../src/utils/tauriApi.js)
  符号：`searchFiles`
  作用：统一分发桌面递归搜索和 SAF 轻量检索。
- 代码位置：[lib.rs](../../../src-tauri/src/lib.rs)
  符号：`search_files`
  作用：桌面端真正执行文件搜索和内容搜索。

## 关键代码怎么读

- 第一步看 [App.jsx](../../../src/App.jsx) 中的 `Ctrl/Cmd+P handler`，其作用是：提供全局搜索弹窗的快捷键入口。
- 第二步看 [SearchModal.jsx](../../../src/components/overlays/SearchModal.jsx) 中的 `SearchModal`，其作用是：负责搜索 UI、模式切换、结果选择和跳转。
- 第三步看 [tauriApi.js](../../../src/utils/tauriApi.js) 中的 `searchFiles`，其作用是：统一分发桌面递归搜索和 SAF 轻量检索。
- 第四步看 [lib.rs](../../../src-tauri/src/lib.rs) 中的 `search_files`，其作用是：桌面端真正执行文件搜索和内容搜索。
