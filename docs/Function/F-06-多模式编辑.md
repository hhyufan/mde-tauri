# 多模式编辑

## 功能是什么

支持源码编辑、只读预览、所见即所得三种模式，部分场景还包含分屏展示。

## 对应代码在哪

- 代码位置：[useEditorStore.js](../../../src/store/useEditorStore.js)
  符号：`setViewMode / toggleEditPreview / toggleSplit`
  作用：负责记录当前是编辑、预览还是分栏模式。
- 代码位置：[EditorContent.jsx](../../../src/components/layout/content/EditorContent.jsx)
  符号：`EditorContent`
  作用：根据当前模式决定渲染 Monaco、Milkdown 还是 MarkdownPreview。
- 代码位置：[TabBar.jsx](../../../src/components/layout/tab-bar/TabBar.jsx)
  符号：`toggleSplit`
  作用：给用户提供切换分栏模式的显式入口。

## 关键代码怎么读

- 第一步看 [useEditorStore.js](../../../src/store/useEditorStore.js) 中的 `setViewMode / toggleEditPreview / toggleSplit`，其作用是：负责记录当前是编辑、预览还是分栏模式。
- 第二步看 [EditorContent.jsx](../../../src/components/layout/content/EditorContent.jsx) 中的 `EditorContent`，其作用是：根据当前模式决定渲染 Monaco、Milkdown 还是 MarkdownPreview。
- 第三步看 [TabBar.jsx](../../../src/components/layout/tab-bar/TabBar.jsx) 中的 `toggleSplit`，其作用是：给用户提供切换分栏模式的显式入口。
