# 多视图一致性

## 功能是什么

编辑区、预览区和大纲视图围绕同一份文档状态协同更新，避免视图间内容漂移。

## 对应代码在哪

- 代码位置：[editorBuffer.js](../../../src/utils/editorBuffer.js)
  符号：`setBuffer / getBuffer / subscribe`
  作用：统一保存正文真值，让多个视图读到同一份内容。
- 代码位置：[useEditorStore.js](../../../src/store/useEditorStore.js)
  符号：`replaceTabContent / markTabDirty / markTabSaved`
  作用：负责标签元信息和内容快照同步。
- 代码位置：[useEditorBufferContent.js](../../../src/hooks/useEditorBufferContent.js)
  符号：`useEditorBufferContent`
  作用：让预览区和大纲视图都能订阅同一份内容变化。

## 关键代码怎么读

- 第一步看 [editorBuffer.js](../../../src/utils/editorBuffer.js) 中的 `setBuffer / getBuffer / subscribe`，其作用是：统一保存正文真值，让多个视图读到同一份内容。
- 第二步看 [useEditorStore.js](../../../src/store/useEditorStore.js) 中的 `replaceTabContent / markTabDirty / markTabSaved`，其作用是：负责标签元信息和内容快照同步。
- 第三步看 [useEditorBufferContent.js](../../../src/hooks/useEditorBufferContent.js) 中的 `useEditorBufferContent`，其作用是：让预览区和大纲视图都能订阅同一份内容变化。
