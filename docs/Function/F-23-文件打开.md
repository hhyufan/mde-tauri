# 文件打开

## 功能是什么

支持打开 Markdown 文件并进入编辑状态。

## 对应代码在哪

- 代码位置：[useFileManager.js](../../../src/hooks/useFileManager.js)
  符号：`openFileDialog / openFileFromPath`
  作用：统一处理打开文件的 UI 入口与实际逻辑。
- 代码位置：[tauriApi.js](../../../src/utils/tauriApi.js)
  符号：`readFileContent`
  作用：统一封装桌面路径与 SAF URI 的读取。
- 代码位置：[lib.rs](../../../src-tauri/src/lib.rs)
  符号：`read_file_content`
  作用：Rust 侧真正完成文本读取并返回编码与换行信息。

## 关键代码怎么读

- 第一步看 [useFileManager.js](../../../src/hooks/useFileManager.js) 中的 `openFileDialog / openFileFromPath`，其作用是：统一处理打开文件的 UI 入口与实际逻辑。
- 第二步看 [tauriApi.js](../../../src/utils/tauriApi.js) 中的 `readFileContent`，其作用是：统一封装桌面路径与 SAF URI 的读取。
- 第三步看 [lib.rs](../../../src-tauri/src/lib.rs) 中的 `read_file_content`，其作用是：Rust 侧真正完成文本读取并返回编码与换行信息。
