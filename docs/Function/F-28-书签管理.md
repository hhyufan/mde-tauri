# 书签管理

## 功能是什么

支持收藏高频文件，并与同步资格判定联动。

## 对应代码在哪

- 代码位置：[useFileStore.js](../../../src/store/useFileStore.js)
  符号：`toggleBookmark / isBookmarked`
  作用：维护书签集合。
- 代码位置：[TabBar.jsx](../../../src/components/layout/tab-bar/TabBar.jsx)
  符号：`handleBookmark`
  作用：在标签栏给当前文档切换书签状态。
- 代码位置：[Sidebar.jsx](../../../src/components/layout/sidebar/Sidebar.jsx)
  符号：`RecentList`
  作用：书签和 recent 混合展示，便于快速进入。

## 关键代码怎么读

- 第一步看 [useFileStore.js](../../../src/store/useFileStore.js) 中的 `toggleBookmark / isBookmarked`，其作用是：维护书签集合。
- 第二步看 [TabBar.jsx](../../../src/components/layout/tab-bar/TabBar.jsx) 中的 `handleBookmark`，其作用是：在标签栏给当前文档切换书签状态。
- 第三步看 [Sidebar.jsx](../../../src/components/layout/sidebar/Sidebar.jsx) 中的 `RecentList`，其作用是：书签和 recent 混合展示，便于快速进入。
