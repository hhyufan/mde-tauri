# 外观设置

## 功能是什么

支持主题、字号、字体、行高等界面和排版配置。

## 对应代码在哪

- 代码位置：[SettingsModal.jsx](../../../src/components/overlays/SettingsModal.jsx)
  符号：`appearance section`
  作用：负责主题、预览字号和缩放等外观设置。
- 代码位置：[useThemeStore.js](../../../src/store/useThemeStore.js)
  符号：`setTheme / initTheme`
  作用：负责主题状态和 `<html data-theme>` 同步。

## 关键代码怎么读

- 第一步看 [SettingsModal.jsx](../../../src/components/overlays/SettingsModal.jsx) 中的 `appearance section`，其作用是：负责主题、预览字号和缩放等外观设置。
- 第二步看 [useThemeStore.js](../../../src/store/useThemeStore.js) 中的 `setTheme / initTheme`，其作用是：负责主题状态和 `<html data-theme>` 同步。
