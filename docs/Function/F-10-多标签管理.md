# 多标签管理

## 功能是什么

支持多文档并行编辑、快速切换，并保留各标签页的编辑状态、保存状态和打开顺序。

## 对应代码在哪

- 代码位置：[useEditorStore.js](../../../src/store/useEditorStore.js)
  符号：`openFile / createUntitledTab / closeTab / setActiveTab`
  作用：负责多标签的创建、切换和关闭。
- 代码位置：[TabBar.jsx](../../../src/components/layout/tab-bar/TabBar.jsx)
  符号：`TabBar`
  作用：负责多标签界面、上下文菜单和重命名入口。
- 代码位置：[editorBuffer.js](../../../src/utils/editorBuffer.js)
  符号：`editorBuffer`
  作用：保证标签切换时未保存内容不会丢失。

## 关键代码怎么读

- 第一步看 [useEditorStore.js](../../../src/store/useEditorStore.js) 中的 `openFile / createUntitledTab / closeTab / setActiveTab`，其作用是：负责多标签的创建、切换和关闭。
- 第二步看 [TabBar.jsx](../../../src/components/layout/tab-bar/TabBar.jsx) 中的 `TabBar`，其作用是：负责多标签界面、上下文菜单和重命名入口。
- 第三步看 [editorBuffer.js](../../../src/utils/editorBuffer.js) 中的 `editorBuffer`，其作用是：保证标签切换时未保存内容不会丢失。
