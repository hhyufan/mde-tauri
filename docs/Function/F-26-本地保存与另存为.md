# 本地保存与另存为

## 功能是什么

支持把文档写回本地，处理已有路径与首次保存两类场景。

## 对应代码在哪

- 代码位置：[useFileManager.js](../../../src/hooks/useFileManager.js)
  符号：`saveTab / saveCurrentFile`
  作用：统一保存入口，区分已落盘、本地未命名和纯云文档。
- 代码位置：[useFileManager.js](../../../src/hooks/useFileManager.js)
  符号：`persistUntitledTab / saveAsDialog`
  作用：处理首次保存和另存为。
- 代码位置：[useFileManager.js](../../../src/hooks/useFileManager.js)
  符号：`finalizeSavedTab`
  作用：保存成功后更新路径、recent、watcher 和同步登记。
- 代码位置：[tauriApi.js](../../../src/utils/tauriApi.js)
  符号：`saveFile`
  作用：统一封装带元信息的保存。
- 代码位置：[lib.rs](../../../src-tauri/src/lib.rs)
  符号：`save_file`
  作用：Rust 侧完成实际写盘。

## 关键代码怎么读

- 第一步看 [useFileManager.js](../../../src/hooks/useFileManager.js) 中的 `saveTab / saveCurrentFile`，其作用是：统一保存入口，区分已落盘、本地未命名和纯云文档。
- 第二步看 [useFileManager.js](../../../src/hooks/useFileManager.js) 中的 `persistUntitledTab / saveAsDialog`，其作用是：处理首次保存和另存为。
- 第三步看 [useFileManager.js](../../../src/hooks/useFileManager.js) 中的 `finalizeSavedTab`，其作用是：保存成功后更新路径、recent、watcher 和同步登记。
- 第四步看 [tauriApi.js](../../../src/utils/tauriApi.js) 中的 `saveFile`，其作用是：统一封装带元信息的保存。
- 最后回看其余支撑文件，补齐该功能的边界条件、状态变化和平台差异。
