# 文件内容搜索

## 功能是什么

可以按照正文内容进行匹配，并定位到命中位置。

## 对应代码在哪

- 代码位置：[SearchModal.jsx](../../../src/components/overlays/SearchModal.jsx)
  符号：`content mode`
  作用：搜索弹窗中按正文内容匹配的前端入口。
- 代码位置：[tauriApi.js](../../../src/utils/tauriApi.js)
  符号：`searchFiles`
  作用：把内容搜索交给桌面 Rust 层，或在 SAF 场景降级。
- 代码位置：[lib.rs](../../../src-tauri/src/lib.rs)
  符号：`search_content_lines`
  作用：桌面端逐行扫描 Markdown 内容并返回命中结果。

## 关键代码怎么读

- 第一步看 [SearchModal.jsx](../../../src/components/overlays/SearchModal.jsx) 中的 `content mode`，其作用是：搜索弹窗中按正文内容匹配的前端入口。
- 第二步看 [tauriApi.js](../../../src/utils/tauriApi.js) 中的 `searchFiles`，其作用是：把内容搜索交给桌面 Rust 层，或在 SAF 场景降级。
- 第三步看 [lib.rs](../../../src-tauri/src/lib.rs) 中的 `search_content_lines`，其作用是：桌面端逐行扫描 Markdown 内容并返回命中结果。
