# 表格与任务列表

## 功能是什么

支持 GFM 表格、任务列表等结构化语法。

## 对应代码在哪

- 代码位置：[MarkdownPreview.jsx](../../../src/components/editor/MarkdownPreview.jsx)
  符号：`components.table / task list rendering`
  作用：预览侧对表格与任务列表做自定义渲染。
- 代码位置：[MilkdownMarkdownEditor.jsx](../../../src/components/editor/MilkdownMarkdownEditor.jsx)
  符号：`taskListCheckboxPlugin / table support`
  作用：WYSIWYG 侧提供表格和任务列表编辑能力。

## 关键代码怎么读

- 第一步看 [MarkdownPreview.jsx](../../../src/components/editor/MarkdownPreview.jsx) 中的 `components.table / task list rendering`，其作用是：预览侧对表格与任务列表做自定义渲染。
