# 文件创建

## 功能是什么

支持在当前工作目录中创建新文件。

## 对应代码在哪

- 代码位置：[useFileManager.js](../../../src/hooks/useFileManager.js)
  符号：`createFileWithDialog`
  作用：在没有工作目录时先选目录，再进入创建流程。
- 代码位置：[useFileManager.js](../../../src/hooks/useFileManager.js)
  符号：`createFileInCurrentDir`
  作用：在当前目录下直接创建空文件并打开。
- 代码位置：[FileTree.jsx](../../../src/components/layout/sidebar/explorer/FileTree.jsx)
  符号：`inline create UI`
  作用：文件树内联新建文件的入口。

## 关键代码怎么读

- 第一步看 [useFileManager.js](../../../src/hooks/useFileManager.js) 中的 `createFileWithDialog`，其作用是：在没有工作目录时先选目录，再进入创建流程。
- 第二步看 [useFileManager.js](../../../src/hooks/useFileManager.js) 中的 `createFileInCurrentDir`，其作用是：在当前目录下直接创建空文件并打开。
- 第三步看 [FileTree.jsx](../../../src/components/layout/sidebar/explorer/FileTree.jsx) 中的 `inline create UI`，其作用是：文件树内联新建文件的入口。
