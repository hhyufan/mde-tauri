# 通用设置

## 功能是什么

支持语言等基础参数配置。

## 对应代码在哪

- 代码位置：[SettingsModal.jsx](../../../src/components/overlays/SettingsModal.jsx)
  符号：`general section`
  作用：负责语言、工作区路径、自动保存等基础设置。
- 代码位置：[useConfigStore.js](../../../src/store/useConfigStore.js)
  符号：`setConfig / loadConfig`
  作用：保存通用设置字段。

## 关键代码怎么读

- 第二步看 [useConfigStore.js](../../../src/store/useConfigStore.js) 中的 `setConfig / loadConfig`，其作用是：保存通用设置字段。
