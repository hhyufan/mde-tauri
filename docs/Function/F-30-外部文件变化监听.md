# 外部文件变化监听

## 功能是什么

监听外部程序对文件的修改，并将变更传播回前端。

## 对应代码在哪

- 代码位置：[useFileManager.js](../../../src/hooks/useFileManager.js)
  符号：`startFileWatching / stopFileWatching`
  作用：在打开、保存、移动后维护文件监听。
- 代码位置：[tauriApi.js](../../../src/utils/tauriApi.js)
  符号：`startFileWatching / onFileChanged`
  作用：前端桥接层负责监听控制和事件订阅。
- 代码位置：[lib.rs](../../../src-tauri/src/lib.rs)
  符号：`start_file_watching / stop_file_watching`
  作用：Rust 侧注册真实文件监听。

## 关键代码怎么读

- 第一步看 [useFileManager.js](../../../src/hooks/useFileManager.js) 中的 `startFileWatching / stopFileWatching`，其作用是：在打开、保存、移动后维护文件监听。
- 第二步看 [tauriApi.js](../../../src/utils/tauriApi.js) 中的 `startFileWatching / onFileChanged`，其作用是：前端桥接层负责监听控制和事件订阅。
- 第三步看 [lib.rs](../../../src-tauri/src/lib.rs) 中的 `start_file_watching / stop_file_watching`，其作用是：Rust 侧注册真实文件监听。
