# 代码块高亮

## 功能是什么

对代码块提供高亮渲染，提升技术文档可读性。

## 对应代码在哪

- 代码位置：[MarkdownPreview.jsx](../../../src/components/editor/MarkdownPreview.jsx)
  符号：`scheduleHighlight / addLangTags`
  作用：预览侧在空闲时对代码块高亮并补语言标签、复制按钮。
- 代码位置：[MarkdownPreview.jsx](../../../src/components/editor/MarkdownPreview.jsx)
  符号：`CodeBlock`
  作用：统一处理普通代码块与 Mermaid 代码块。
- 代码位置：[MilkdownMarkdownEditor.jsx](../../../src/components/editor/MilkdownMarkdownEditor.jsx)
  符号：`enhanceCodeBlocks`
  作用：WYSIWYG 只读渲染时增强代码块展示效果。

## 关键代码怎么读

- 第一步看 [MarkdownPreview.jsx](../../../src/components/editor/MarkdownPreview.jsx) 中的 `scheduleHighlight / addLangTags`，其作用是：预览侧在空闲时对代码块高亮并补语言标签、复制按钮。
- 第二步看 [MarkdownPreview.jsx](../../../src/components/editor/MarkdownPreview.jsx) 中的 `CodeBlock`，其作用是：统一处理普通代码块与 Mermaid 代码块。
- 第三步看 [MilkdownMarkdownEditor.jsx](../../../src/components/editor/MilkdownMarkdownEditor.jsx) 中的 `enhanceCodeBlocks`，其作用是：WYSIWYG 只读渲染时增强代码块展示效果。
