# 云同步设置

## 功能是什么

支持服务器地址、同步开关等与云端能力相关的参数。

## 对应代码在哪

- 代码位置：[SettingsModal.jsx](../../../src/components/overlays/SettingsModal.jsx)
  符号：`sync section / handleSyncSettings / handlePullSettings`
  作用：负责同步开关、服务器地址、手动推送和拉取设置。
- 代码位置：[syncEngine.js](../../../src/services/syncEngine.js)
  符号：`syncConfig`
  作用：负责把设置真正和服务端对齐。

## 关键代码怎么读

- 第一步看 [SettingsModal.jsx](../../../src/components/overlays/SettingsModal.jsx) 中的 `sync section / handleSyncSettings / handlePullSettings`，其作用是：负责同步开关、服务器地址、手动推送和拉取设置。
- 第二步看 [syncEngine.js](../../../src/services/syncEngine.js) 中的 `syncConfig`，其作用是：负责把设置真正和服务端对齐。
