# 延迟自动保存

## 功能是什么

在用户暂停输入后自动把内容写回本地磁盘，降低内容丢失风险。

## 对应代码在哪

- 代码位置：[useConfigStore.js](../../../src/store/useConfigStore.js)
  符号：`autoSave`
  作用：自动保存的总开关配置。
- 代码位置：[MonacoEditor.jsx](../../../src/components/editor/MonacoEditor.jsx)
  符号：`scheduleAutoSave`
  作用：源码模式输入后节流触发保存。
- 代码位置：[MilkdownMarkdownEditor.jsx](../../../src/components/editor/MilkdownMarkdownEditor.jsx)
  符号：`saveTimerRef`
  作用：WYSIWYG 模式输入后节流触发保存。
- 代码位置：[useFileManager.js](../../../src/hooks/useFileManager.js)
  符号：`triggerAutoSave / debouncedAutoSave`
  作用：统一执行本地落盘或外部文档同步。

## 关键代码怎么读

- 第一步看 [useConfigStore.js](../../../src/store/useConfigStore.js) 中的 `autoSave`，其作用是：自动保存的总开关配置。
- 第二步看 [MonacoEditor.jsx](../../../src/components/editor/MonacoEditor.jsx) 中的 `scheduleAutoSave`，其作用是：源码模式输入后节流触发保存。
- 第三步看 [MilkdownMarkdownEditor.jsx](../../../src/components/editor/MilkdownMarkdownEditor.jsx) 中的 `saveTimerRef`，其作用是：WYSIWYG 模式输入后节流触发保存。
- 第四步看 [useFileManager.js](../../../src/hooks/useFileManager.js) 中的 `triggerAutoSave / debouncedAutoSave`，其作用是：统一执行本地落盘或外部文档同步。
