# 跨平台运行

## 功能是什么

系统面向桌面端与 Android 端提供统一核心编辑能力。

## 对应代码在哪

- 代码位置：[platform.js](../../../src/utils/platform.js)
  符号：`isMobile / isAndroid / isDesktopWindow`
  作用：统一输出当前运行平台信息。
- 代码位置：[useResponsiveLayout.js](../../../src/hooks/useResponsiveLayout.js)
  符号：`useResponsiveLayout`
  作用：输出是否小屏、是否竖屏等布局判定。
- 代码位置：[lib.rs](../../../src-tauri/src/lib.rs)
  符号：`run`
  作用：桌面端和 Android 端共用的 Tauri 应用入口。
- 代码位置：[tauri.conf.json](../../../src-tauri/tauri.conf.json)
  符号：`bundle / app config`
  作用：定义跨平台打包和窗口行为。

## 关键代码怎么读

- 第一步看 [platform.js](../../../src/utils/platform.js) 中的 `isMobile / isAndroid / isDesktopWindow`，其作用是：统一输出当前运行平台信息。
- 第二步看 [useResponsiveLayout.js](../../../src/hooks/useResponsiveLayout.js) 中的 `useResponsiveLayout`，其作用是：输出是否小屏、是否竖屏等布局判定。
- 第三步看 [lib.rs](../../../src-tauri/src/lib.rs) 中的 `run`，其作用是：桌面端和 Android 端共用的 Tauri 应用入口。
- 第四步看 [tauri.conf.json](../../../src-tauri/tauri.conf.json) 中的 `bundle / app config`，其作用是：定义跨平台打包和窗口行为。
