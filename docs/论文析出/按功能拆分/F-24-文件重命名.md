# 文件重命名

## 功能是什么

支持在文件管理流程中重命名文档。

## 对应代码在哪

- 代码位置：[TabBar.jsx](../../../src/components/layout/tab-bar/TabBar.jsx)
  符号：`commitRename`
  作用：标签重命名时同步更新真实文件路径。
- 代码位置：[tauriApi.js](../../../src/utils/tauriApi.js)
  符号：`renameFile`
  作用：统一封装桌面重命名与 SAF 改名。
- 代码位置：[lib.rs](../../../src-tauri/src/lib.rs)
  符号：`rename_file`
  作用：Rust 侧执行真实重命名或移动。

## 关键代码怎么读

- 第一步看 [TabBar.jsx](../../../src/components/layout/tab-bar/TabBar.jsx) 中的 `commitRename`，其作用是：标签重命名时同步更新真实文件路径。
- 第二步看 [tauriApi.js](../../../src/utils/tauriApi.js) 中的 `renameFile`，其作用是：统一封装桌面重命名与 SAF 改名。
- 第三步看 [lib.rs](../../../src-tauri/src/lib.rs) 中的 `rename_file`，其作用是：Rust 侧执行真实重命名或移动。
