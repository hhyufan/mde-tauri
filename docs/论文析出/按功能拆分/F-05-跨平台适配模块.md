# 跨平台适配模块

## 功能是什么

面向桌面端与 Android 端的差异做兼容与补充处理。

## 对应代码在哪

- 代码位置：[tauriApi.js](../../../src/utils/tauriApi.js)
  符号：`tauriApi`
  作用：负责对桌面能力和 Android 文件能力做统一封装。
- 代码位置：[androidSaf.js](../../../src/utils/androidSaf.js)
  符号：`androidSaf`
  作用：负责前端 Android SAF 桥接。
- 代码位置：[MainActivity.kt](../../../src-tauri/gen/android/app/src/main/java/com/mde/app/MainActivity.kt)
  符号：`MainActivity`
  作用：负责 Android 端选择器注册和桥接注入。
- 代码位置：[SafBridge.kt](../../../src-tauri/gen/android/app/src/main/java/com/mde/app/SafBridge.kt)
  符号：`SafBridge`
  作用：负责 Android 端真正的目录访问与文件读写。
- 代码位置：[useResponsiveLayout.js](../../../src/hooks/useResponsiveLayout.js)
  符号：`useResponsiveLayout`
  作用：负责移动端布局判定。
- 代码位置：[useViewportInsets.js](../../../src/hooks/useViewportInsets.js)
  符号：`useViewportInsets`
  作用：负责安全区与软键盘遮挡处理。
