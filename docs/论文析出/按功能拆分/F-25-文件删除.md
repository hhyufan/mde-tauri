# 文件删除

## 功能是什么

支持删除文件，是文件管理基础能力之一。

## 对应代码在哪

- 代码位置：[FileTree.jsx](../../../src/components/layout/sidebar/explorer/FileTree.jsx)
  符号：`delete action`
  作用：文件树中触发删除。
- 代码位置：[tauriApi.js](../../../src/utils/tauriApi.js)
  符号：`deleteFile`
  作用：统一封装桌面与 SAF 删除。
- 代码位置：[lib.rs](../../../src-tauri/src/lib.rs)
  符号：`delete_file`
  作用：Rust 侧执行文件或目录删除。

## 关键代码怎么读

- 第一步看 [FileTree.jsx](../../../src/components/layout/sidebar/explorer/FileTree.jsx) 中的 `delete action`，其作用是：文件树中触发删除。
- 第二步看 [tauriApi.js](../../../src/utils/tauriApi.js) 中的 `deleteFile`，其作用是：统一封装桌面与 SAF 删除。
- 第三步看 [lib.rs](../../../src-tauri/src/lib.rs) 中的 `delete_file`，其作用是：Rust 侧执行文件或目录删除。
