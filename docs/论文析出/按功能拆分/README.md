# 按功能讲解的代码文档索引

这一套文档不再讨论“问题是什么”，而是按功能讲“这个功能在哪里实现、先看哪几段代码、这些代码怎么协作”。

## 使用方法

- 想看全局，先读 `F-01` 到 `F-05` 这 5 个一级模块。
- 想看某个具体能力，直接进入对应功能名的文件。
- 想继续深挖源码，可以从每个文档里的“关键代码怎么读”顺序继续往下追。
