# 文件树浏览

## 功能是什么

支持目录展开与折叠，形成资源管理器式浏览体验。

## 对应代码在哪

- 代码位置：[FileTree.jsx](../../../src/components/layout/sidebar/explorer/FileTree.jsx)
  符号：`FileTree`
  作用：文件树主组件，负责目录导航、排序和点击行为。
- 代码位置：[Sidebar.jsx](../../../src/components/layout/sidebar/Sidebar.jsx)
  符号：`ExplorerToolbar`
  作用：负责文件树上方的排序、新建、保存、打开目录入口。
- 代码位置：[useFileStore.js](../../../src/store/useFileStore.js)
  符号：`setSortMode / files`
  作用：负责文件树展示所需的数据与排序状态。

## 关键代码怎么读

- 第一步看 [FileTree.jsx](../../../src/components/layout/sidebar/explorer/FileTree.jsx) 中的 `FileTree`，其作用是：文件树主组件，负责目录导航、排序和点击行为。
- 第二步看 [Sidebar.jsx](../../../src/components/layout/sidebar/Sidebar.jsx) 中的 `ExplorerToolbar`，其作用是：负责文件树上方的排序、新建、保存、打开目录入口。
- 第三步看 [useFileStore.js](../../../src/store/useFileStore.js) 中的 `setSortMode / files`，其作用是：负责文件树展示所需的数据与排序状态。
