# 工作目录加载

## 功能是什么

加载工作目录并展示为文件树，是文件管理的入口能力。

## 对应代码在哪

- 代码位置：[useFileStore.js](../../../src/store/useFileStore.js)
  符号：`currentDir / setCurrentDir`
  作用：负责保存当前工作目录及目录历史。
- 代码位置：[useFileManager.js](../../../src/hooks/useFileManager.js)
  符号：`loadDirectory / openFolderDialog`
  作用：负责选择工作目录并加载为资源管理器数据源。
- 代码位置：[lib.rs](../../../src-tauri/src/lib.rs)
  符号：`get_directory_contents`
  作用：Rust 侧返回目录直接子项。

## 关键代码怎么读

- 第一步看 [useFileStore.js](../../../src/store/useFileStore.js) 中的 `currentDir / setCurrentDir`，其作用是：负责保存当前工作目录及目录历史。
- 第二步看 [useFileManager.js](../../../src/hooks/useFileManager.js) 中的 `loadDirectory / openFolderDialog`，其作用是：负责选择工作目录并加载为资源管理器数据源。
- 第三步看 [lib.rs](../../../src-tauri/src/lib.rs) 中的 `get_directory_contents`，其作用是：Rust 侧返回目录直接子项。
